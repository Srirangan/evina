    minimalist in-app events library

    Introduction

    Evina is a JavaScript eventing library. It was created for in-browser usage.

    Usage

    <script src="evina.min.js"></script>
<script>
    evina.on("some-event", function (context) {
        console.log("some event happened");
        console.log("and we have", context);
    });

    evina.trigger("some-event", {hello: "world"});

    evina.off("some-event");
</script>
